# CTE Mental Wellness Manuscript Review

Dark-mode elegant webapp for co-author review of manuscript contribution materials.

## Study

**Qualitative Study Examining CTE Health Science Pathway Students' Involvement in Practicing and Promoting Mental Wellness**

- **Original Study:** 2020
- **Co-Authors:** Charles, Marie, Loera G., Nakamoto J.
- **Target Journals:** Journal of Research in Technical Careers, Journal of Urban Education

## Features

- 5 content cards with modal views
- Print-optimized APA7 formatting
- Dark mode glassmorphism design
- Fully responsive layout
- Keyboard accessible (Tab, Enter, Escape)

## Content Sections

1. **Practice Implications** - New section connecting study themes to AI-augmented healthcare
2. **Literature Review Updates** - 5 new paragraphs with 2020-2026 citations
3. **APA7 References** - 27 new references with verification status
4. **Edit Notes** - Section-by-section checklist including Cal-HOSA terminology removal
5. **Project Overview** - Study summary and next steps

## GitHub Pages Deployment

### Option 1: New Repository

1. Create new repo: `cte-manuscript-review`
2. Upload all files to root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `print.css`
   - `README.md`
3. Go to Settings > Pages
4. Source: Deploy from branch
5. Branch: `main` / `root`
6. Save and wait for deployment

### Option 2: Existing Repo Subfolder

1. Create folder `/cte-manuscript/` in existing repo
2. Add all files
3. Access at: `https://username.github.io/repo-name/cte-manuscript/`

## Local Preview

Simply open `index.html` in a browser. No build step required.

## Print Instructions

1. Open any content card
2. Click "Print Section" button
3. Print dialog will open with APA7-formatted content
4. Save as PDF or print directly
